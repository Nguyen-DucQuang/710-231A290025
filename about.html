Giới Thiệu

Chào mừng đến với trang web Bài Tập Thực Hành

Đây là dự án tổng hợp các bài tập thực hành môn Lập trình Web Front-end, được phát triển bởi sinh viên lớp 710 với MSSV tương ứng. Trang web được xây dựng nhằm mục đích học tập và rèn luyện kỹ năng phát triển web.

Tuần phát triển

Sứ mệnh

Cung cấp một nền tảng học tập trực quan và thực tế cho sinh viên công nghệ thông tin, giúp nâng cao kỹ năng lập trình web thông qua các bài tập thực hành có tính ứng dụng cao.

Tầm nhìn

Trở thành nguồn tài liệu tham khảo hữu ích cho sinh viên các khóa sau, đồng thời phát triển thành một dự án mã nguồn mở có thể đóng góp vào cộng đồng học lập trình web.

Tính năng nổi bật

Code sạch đẹp

Sử dụng HTML5 semantic, CSS3 hiện đại và JavaScript ES6+ với cấu trúc rõ ràng, dễ bảo trì.

Responsive Design

Tối ưu hiển thị trên mọi thiết bị từ mobile đến desktop với Flexbox và Grid layout.

Giao diện hiện đại

Thiết kế giao diện đẹp mắt với gradient, animation và hiệu ứng chuyển động mượt mà.

Bảo mật tốt

Áp dụng các nguyên tắc bảo mật cơ bản như input validation, XSS prevention.

Công nghệ sử dụng

HTML5 CSS3 JavaScript ES6+ LocalStorage Font Awesome
